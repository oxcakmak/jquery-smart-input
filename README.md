# jquery-smart-input
jquery smart input

 
I made qjuery based smart input that you can use in your projects.

So how is this code different from the others?
- When the number you enter in an input is full, it automatically switches to the other input.
- It automatically unfocuses when all fields are full (so you can click the button).
- When you get an error, it allows you to automatically delete the codes by refocusing.
- If you press delete on the second input, it will delete step by step from the last to the first.

!: you can increase the number of inputs if you want, but with one condition; only if there is a “digit-input” in the input class (you can change it yourself if you want)

Projelerinizde kullanabileceğiniz qjuery bazlı smart input yaptım.

Peki bu kodun diğerlerininden ne farkı var?
* Bir inputa girdiğiniz numara dolduğunda otomatik diğer inputa geçiş yapar.
* Tüm alanlar dolduğunda otomatik unfokus yapıyor (yani butona tıklayabiliyorsunuz).
* Hata aldığınzda tekrar fokus atarak otomatik olarak kodları silmenizi sağlar.
* Eğer ikinci inputta delete tuşuna bastığınızda en sondan ilk başa doğru adım adım silme işlemi gerçekleştirir.

!: isterseniz input sayısını çoğaltabilirsiniz ama bir şartla; sadece input class içerisinde "digit-input" bulunması şartıyla (isteğe bağlı kendiniz de değiştirebilirsiniz)
